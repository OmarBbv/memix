# 🗂 Memix API - Modullar

Hal-hazırda aşağıdakı modullar aktivdir:

1.  **👤 Users** - İstifadəçilər, ünvanlar və rollar.
2.  **🔐 Auth** - Giriş, qeydiyyat, JWT və Google ilə giriş.
3.  **📦 Products** - Məhsullar, axtarış və filtrləmə.
4.  **📂 Categories** - Kateqoriyalar və dinamik filtrlər.
5.  **🏢 Branches** - Filiallar və stok idarəetməsi.
6.  **🛒 Carts** - Səbət (alış-veriş səbəti).
7.  **🚚 Orders** - Sifarişlər, statuslar və çatdırılma.
8.  **🖼️ Common (Uploads)** - Şəkil yükləmə və köməkçi fayllar.
